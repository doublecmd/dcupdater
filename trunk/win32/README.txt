- Installation:
Just extract DCUpdater.exe into Double Commander root directory. DCUpdater
have to be located in the same folder as doublecmd.exe.
For instance: C:\Program Files\doublecmd

- Usage:
Execute DCUpdater.exe

Make sure that Double Commander is not already running. DCUpdater will skip
the update process if Double Commander is already running.
